## À propos

Développeur web full-stack reconverti en intégrateur e-learning, je produis des modules de formation numérique interactifs pour des organismes de formation, des services RH et des entreprises souhaitant digitaliser leurs contenus pédagogiques.

Ce dépôt contient mon portfolio public : des modules e-learning entièrement réalisés en HTML5 natif, ainsi que des productions Articulate Rise 360 exportées au format Web.

Mon positionnement repose sur un **double profil technique et pédagogique** : maîtrise des outils de production (Rise 360, HTML5, SCORM) et formation universitaire en ingénierie de la formation à distance (L3 Sciences de l'Éducation — campus FORSE, Université Lyon 2 / Rouen Normandie).

## 🛠️ Stack technique

| Technologie | Usage |
|---|---|
| HTML5 | Structure et sémantique des modules |
| CSS3 | Mise en page, animations, responsive design |
| JavaScript vanilla | Interactions, quiz, score, navigation entre écrans |
| Google Fonts | Nunito, Syne, DM Sans |
| Articulate Rise 360 | Production modules e-learning (export Web + SCORM 1.2) |

Aucune dépendance externe, aucun framework. Tous les modules fonctionnent en ouvrant le fichier HTML directement dans un navigateur.

## 📦 Formats de livraison

| Format | Usage |
|---|---|
| **URL publique** (GitHub Pages) | Démonstration, portfolio, partage client |
| **Export Web** (Rise 360) | Hébergement sur serveur ou LMS sans tracking |
| **SCORM 1.2** (.zip) | Intégration sur LMS (Moodle, 360Learning, Talentsoft…) avec suivi des scores et de la complétion |
